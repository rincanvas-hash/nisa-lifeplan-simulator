# 50代からの未来設計シミュレーター

Python と Streamlit で作成した、日本語の老後資金シミュレーターです。サブタイトルは「老後資金の見通しを、数字でやさしく確認」です。現在の金融資産、退職前の毎月の積立額、想定利回り、退職後の生活費と年金収入を入力すると、年齢ごとの資産残高の目安を確認できます。

## 第2版の画面構成

初心者でも順番に進められるよう、`st.Page` と `st.navigation` を使った 3 ステップ構成にしています。3ページすべての上部に、統一した大きなアプリ名、サブタイトル、ページ名を表示します。

1. **はじめに**
   - アプリの目的を短く説明します。
   - 教育目的の概算であり、金融助言ではないことを表示します。
   - 入力内容は保存されないことを表示します。
   - 「シミュレーションを始める」ボタンから入力へ進みます。
2. **条件を入力**
   - 入力欄を左サイドバーではなくメイン画面に配置しています。
   - 年齢と想定利回りは `number_input` で入力します。
   - 金額は `text_input` で入力し、カンマ、空白、円記号を含む入力も数値へ変換します。
   - 数字以外が含まれる場合は、日本語で分かりやすいエラーを表示します。
3. **結果**
   - 退職時の推定資産、退職後の毎月不足額、資産がなくなる推定年齢、終了年齢での推定資産を表示します。
   - 資産推移グラフと年間結果表を表示します。
   - 計算の前提と注意事項を表示します。
   - 「条件を変更する」ボタンで入力ページへ戻れます。

## できること

- 退職時の推定資産を表示
- 退職後の毎月不足額を表示
- 資産がなくなる推定年齢を表示
- シミュレーション終了年齢での推定資産を表示
- 年齢ごとの資産残高グラフを表示
- かんたんな年間結果表を表示

## 計算の前提

このアプリは MVP の第2版として、次のような簡略化した計算を行います。

- 退職前は、現在の資産に毎月の積立額と想定利回りを反映します。
- 退職後は、毎月の生活費から毎月の年金収入を差し引いた不足額を資産から取り崩します。
- 資産が残っている間は、退職後も同じ想定利回りを反映します。
- 資産残高は 0 円未満として表示しません。
- 税金、社会保険料、物価上昇、退職金、夫婦での計算は含めていません。

## 大切な注意事項

- このシミュレーターは教育目的の概算ツールです。
- 金融助言ではありません。
- 結果は将来を保証するものではありません。
- 入力された個人情報や数値は保存しません。
- 想定利回りを高くしすぎると、現実より楽観的な結果になる可能性があります。

## セットアップ

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 起動方法

```bash
streamlit run app.py
```

ブラウザで表示された URL を開いて利用してください。公開 URL と `app.py` のエントリーポイントは従来どおりです。
