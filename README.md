# 50代からの老後資金かんたんシミュレーター

Python と Streamlit で作成した、日本語の老後資金シミュレーターです。現在の金融資産、退職前の毎月の積立額、想定利回り、退職後の生活費と年金収入を入力すると、年齢ごとの資産残高の目安を確認できます。

## できること

- 退職時の推定資産を表示
- 退職後の毎月不足額を表示
- 資産がなくなる推定年齢を表示
- シミュレーション終了年齢での推定資産を表示
- 年齢ごとの資産残高グラフを表示
- かんたんな年間結果表を表示

## 計算の前提

このアプリは最初の MVP 版として、次のような簡略化した計算を行います。

- 退職前は、現在の資産に毎月の積立額と想定利回りを反映します。
- 退職後は、毎月の生活費から毎月の年金収入を差し引いた不足額を資産から取り崩します。
- 資産が残っている間は、退職後も同じ想定利回りを反映します。
- 資産残高は 0 円未満として表示しません。
- 税金、社会保険料、物価上昇、退職金、夫婦での計算は含めていません。

## 大切な注意事項

- このシミュレーターは教育目的の概算ツールです。
- 金融助言ではありません。
- 結果は将来を保証するものではありません。
- 入力された個人情報や数値は保存しません。
- 想定利回りを高くしすぎると、現実より楽観的な結果になる可能性があります。

## セットアップ

```bash
python -m venv .venv

pip install -r requirements.txt
```

```bash
streamlit run app.py
```
